1. 개인정보 처리의 목적

모리의 여행플래너(이하 '서비스')는 사용자의 여행 계획 관리 서비스를 제공하기 위해 개인정보를 처리합니다. 본 방침은 이용자의 개인정보 보호를 위해 작성되었으며, 서비스 이용 중 발생할 수 있는 개인정보 처리에 대하여 설명합니다.

2. 수집하는 개인정보

2-1. Google 계정을 통한 수집

서비스는 Google OAuth를 통해 다음 정보를 수집합니다:

  • Google 계정 ID: 사용자 식별용
  • 기본 프로필 정보: 이름, 이메일 주소, 프로필 사진 (선택)
  • Google Drive 접근 권한: 여행 데이터 저장/복구용
보안 참고: OAuth 토큰은 현재 탭에서만 유효하며, 탭을 닫으면 자동으로 소실됩니다. 브라우저 종료 후 다시 로그인이 필요합니다.

2-2. 로컬 저장 데이터

사용자의 브라우저 로컬스토리지에 저장되는 데이터:

데이터 항목 저장 위치 설명
여행 일정 로컬스토리지 여행 이름, 날짜, 일정, 위치, 메모 등
지출 내역 로컬스토리지 금액, 통화, 분류, 결제수단
체크리스트 로컬스토리지 준비 항목, 완료 상태
설정 정보 로컬스토리지 테마, 언어, 알림 설정
여행 기록 로컬스토리지 지난 여행 아카이브

2-3. 기술적 정보

  • 쿠키 및 캐시: 서비스 성능 최적화용 (서비스워커 캐시)
  • 환경 정보: 브라우저 종류, 해상도, OS 등 (서비스 호환성 확인용)

개인정보의 제3자 공개 불가

사용자의 여행 데이터는 전적으로 개별 사용자의 로컬스토리지 또는 개별 Google Drive에만 저장됩니다. 서비스는 다음을 보장합니다:

  • 사용자 데이터를 기업 서버에 저장하지 않음
  • 제3의 광고 회사 또는 마케팅 업체에 제공하지 않음
  • 법적 요청이 없는 한 정부기관에 제공하지 않음

4. 개인정보의 보관 기간

4-1. 로컬스토리지 데이터

  • 보관 기간: 사용자가 브라우저 데이터를 삭제할 때까지
  • 자동 삭제: 없음 (사용자가 수동으로 삭제하거나 '여행 마무리' 기능으로 삭제)

4-2. Google Drive 백업

  • 보관 기간: Google Drive 계정이 유지되는 동안
  • 삭제 방법: Google Drive에서 수동 삭제 또는 Google 계정 삭제

4-3. OAuth 토큰

  • 보관 기간: 브라우저 탭이 열려있는 동안만
  • 자동 소실: 탭 닫음 시 즉시 소실
주의: 서비스 이용 중단 시 저장된 데이터는 자동 삭제되지 않습니다. 필요시 사용자가 수동으로 삭제해야 합니다.

5. 제3자에게의 정보 제공

5-1. 정보 제공 현황

모리의 여행플래너는 사용자 동의 없이 제3자에게 개인정보를 제공하지 않습니다.

5-2. 필수 파트너

다음의 경우에만 정보 공개가 발생합니다:

  • Google LLC: OAuth 및 Google Drive 서비스 제공 (사용자 선택)
  • GitHub Pages: 웹 호스팅 (서비스 배포용)

이들 서비스는 각각의 개인정보보호방침을 따릅니다:

  • Google 개인정보보호방침
  • GitHub 개인정보보호방침

5-3. 법적 요청

법원의 명령, 수사기관의 요청 등 법률에 따른 의무가 있는 경우에만 정보를 제공할 수 있습니다.

6. 개인정보 보호 조치

6-1. 기술적 보호 조치

  • HTTPS 암호화: 모든 통신은 SSL/TLS 암호화됨
  • 로컬스토리지 격리: 브라우저의 동일 출처 정책(Same-Origin Policy)으로 보호
  • Google Drive 보안: Google의 엔드-투-엔드 암호화 및 접근 제어
  • 서비스워커 캐시: 오프라인 기능 제공 및 성능 최적화

6-2. 접근 제어

  • Google OAuth를 통한 본인 확인만이 데이터 접근 허용
  • 로컬스토리지는 현재 도메인(simbaking84.github.io)에서만 접근 가능
  • 토큰 기반 인증으로 무단 접근 방지

6-3. 주의 사항

사용자는 다음 사항을 반드시 지켜주세요:

  • 공용 컴퓨터에서는 사용 후 반드시 로그아웃
  • Google 계정의 보안을 유지 (강력한 비밀번호, 2단계 인증)
  • 피싱 이메일 및 의심스러운 링크를 피하기

7. 사용자의 권리

7-1. 개인정보 열람 및 수정 요청

사용자는 다음의 권리를 행사할 수 있습니다:

  • 열람: 로컬스토리지 및 Google Drive에 저장된 자신의 데이터 확인
  • 수정: 서비스 내 편집 기능으로 데이터 직접 수정
  • 삭제: '여행 마무리' 또는 브라우저 데이터 삭제로 로컬 데이터 제거
  • Google Drive 삭제: Google Drive에서 직접 백업 파일 삭제 가능

7-2. 데이터 이동권

  • JSON 형식으로 데이터 내보내기 기능 제공
  • 엑셀(CSV) 형식으로 데이터 다운로드 가능
  • 다른 서비스로의 데이터 이전 용이

7-3. 동의 철회

  • Google Drive 연동: 설정에서 '계정 연결 해제' 버튼으로 언제든 철회 가능
  • 서비스 이용 중단: 언제든 계정 삭제 가능 (데이터는 로컬에서 삭제됨)

8. 개인정보처리방침의 변경

서비스는 법령의 변경에 따라 본 방침을 변경할 수 있습니다. 변경 사항은 다음과 같이 공지합니다:

8-1. 변경 공지 방법

  • 웹사이트 공지: 본 페이지의 '최종 수정' 날짜 업데이트
  • 주요 변경: 서비스 내 공지사항 또는 이메일 안내

8-2. 변경 효력 발생

  • 경미한 변경: 즉시 적용
  • 중요한 변경: 최소 30일 전 공지 후 적용
  • 사용자의 권리를 제한하는 변경: 사전 동의 필수
변경 이력 확인: 이 페이지를 주기적으로 확인하여 최신 정책을 확인하시기 바랍니다.
